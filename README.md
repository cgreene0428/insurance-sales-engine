# Insurance Sales Engine

**Full-lifecycle CRM buildout for P&C insurance agencies** built for AgencyZoom, powered by the Trust and Grow Framework.

Built by **Chris Greene** at [Niche Your Business](https://www.buildinganiche.com).

## Installation

In Claude Code CLI:
```
/plugin marketplace add cgreene0428/insurance-sales-engine
/plugin install insurance-sales-engine@insurance-sales-engine
```

In Claude Desktop (Cowork):
1. Go to Settings, Plugins
2. Click Add Marketplace
3. Enter: cgreene0428/insurance-sales-engine
4. Find Insurance Sales Engine and click Install
